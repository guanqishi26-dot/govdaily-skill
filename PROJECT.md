# GovDaily Skill

## Project Vision

GovDaily Skill is an AI-powered pipeline that transforms official news into exam-oriented daily reports for Chinese Civil Service Exam preparation.

---

## Target Users

Primary users:

- Developers preparing for the Civil Service Exam
- AI enthusiasts
- Users who prefer automated daily briefings

---

## MVP (v0.1)

The first version only needs to:

- Fetch RSS feeds
- Select 5–8 important news
- Generate exam tags
- Generate key insights
- Export one HTML report

Nothing more.

---
